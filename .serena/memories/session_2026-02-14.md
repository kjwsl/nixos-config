## Status Update - Later in Session

### Issues Encountered After Activation
1. ❌ tmux-powerkit failed with exit code 1
2. ❌ tmux opened zsh instead of fish

### Root Causes Identified
1. **Bash Version Incompatibility**:
   - tmux-powerkit requires Bash 5.1+ 
   - macOS ships with Bash 3.2.57
   - Plugin shebang used system bash
   
2. **Shell Configuration Override**:
   - home-manager auto-adds `reattach-to-user-namespace` on macOS
   - Auto-generated `default-command` hardcoded `/bin/zsh`
   - `default-command` overrides `default-shell`

### Solutions Applied
1. **Fixed tmux-powerkit**: Patched shebang to use Nix bash 5.3
2. **Fixed fish shell**: Added `shell = "${pkgs.fish}/bin/fish"` + override `default-command`

See detailed analysis: `session_2026-02-14_tmux-powerkit-fix.md`
