# Mistake: Over-engineering Instead of Using Framework

## What Happened (2026-02-14)
User shared tmux-powerkit reference. I built custom config instead of using it directly.

## Why It Failed
- Assumed "like X" meant "build similar" not "use X"
- Manual plugins vs comprehensive framework
- Result: "looks ugly and shit"

## Correction
User asked: "why can't we just use tmux-powerkit?"
Answer: We absolutely should!

## Lesson
When comprehensive framework exists, USE IT. Don't reinvent.
Ask: "Do you want to USE this or build something LIKE this?"
